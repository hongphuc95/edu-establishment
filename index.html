Bienvenue

Présentation

Bienvenue, ce site a été construit par Hong Phuc Vu pour le projet de Développement Web de licence Informatique en deuxième année. Profitez-vous bien sur nos pages. Si vous avez des questions, n'hésitez pas à aller dans la rubrique "Contacter". Nous sommes toujours disponibles pour vous aider.

Sujet

A partir des données publiques des établissements d’enseignement supérieur en France recensés par l’ONISEP (métropole et outre-mer), on souhaite créer un site web de visualisation / mise en valeur / recherche dans ce jeu de données.
